Trngle is an app mean’t to network creatives and allow for them to collaborate on projects. Trngle is also mean’t to promote these projects and attract both attention and talent.

The App

Login

The login screen provides the option to sign in with Facebook as well as register an account. After registering the user is taken to the account creation page.

Project Listings

Once an account is created and a user is logged in, the primary home screen become the project listings page. Each project on this page uses a photo to set the tone and quickly provides context to the user for the type of project. Below each project are the roles the creator of the project is looking to fill. Next to those are the time and location that the project will occupy. Actioning the top bar of the page will, as indicated by the arrow, provide a drop down menu for settings. This page allows the user to browse active projects and engage with them by up voting them or accessing the project poster's profile or the details of the project itself.

Project Page

The project page provides general details about the project and also specific information about what is desired from each of the positions applicants. The positions display the number of taken and available slots which shows the user how soon the project is from establising a team and beginning. Once a project is filled, the position information is replaced by a group message for the team to discuss.

Menu

I implimented a hamburger menu due to the complexity of the app and the number of features and pages. The post action is highlighted and encourages users to create projects.

Profile

The profile page allows users to add work, which acts as a type of in-app portfolio. When there are no past Trngle projects to populate this part, a user may add links to external work themselves. Users add work in the settings page. When a user has applied to that project, the project owner can accept or decline the applicant on their profile.

Account Activity

The account activity page keeps a user updated on the happenings of all their projects. Tabs are used to filter between all acount activity, activity on the user’s projects, and activity on the projects they have joined. All provides a list of recent relevant communications and user actions. Projects provides updates on you application status, and also links to those projects. Your projects breaks down the account activity on a per project basis.
