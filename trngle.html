Project Feed

Trngle is an app meant to network creatives and enable them to collaborate on projects. I worked with the Trngle team to develop the app’s UI. The project feed is for discovering projects to engage with and can be filtered to display work’ve relevant the skills users have listed. A user may engage with each project by up-voting or choosing to apply for a position on the project’s team. Actioning the top bar of the page provides a drop down menu for filtering through the projects.

Project Page

The project page provides general details about the project and also specific information about what is desired from each of the positions applicants. The positions display the number of taken and available slots which show the user how soon the project is from establishing a team and beginning. Once a project is filled, the position information is replaced by a group message for the team to discuss.

Menu

I chose to implement a hamburger menu due to the complexity of the app and the number of features and pages. The post action is highlighted and encourages users to create projects.

Account Activity

The account activity page keeps a user updated on the happenings of all their projects. Tabs are used to filter between all account activity, activity on the user’s projects, and activity on the projects they have joined. All provides a list of recent relevant communications and user actions. Projects provides updates on a user’s application status, and also links to those projects. Your projects breaks down the account activity on a per project basis.
